## 特徴

<CardGrid stagger>
  <Card title="Intrinsic Design" icon="rocket">
    メディアクエリに頼らず、CSSの本来の力（flexbox、grid、CSS Custom Properties）を活用した設計。
  </Card>
  <Card title="コンポジション" icon="puzzle">
    シンプルなプリミティブを組み合わせて、複雑なレイアウトを構築。
  </Card>
  <Card title="TypeScript対応" icon="document">
    完全な型定義付き。エディタでの補完やエラーチェックをサポート。
  </Card>
  <Card title="Tree-shakable" icon="seti:folder">
    コンポーネントごとにCSSが分離されており、使用するものだけがバンドルに含まれます。
  </Card>
</CardGrid>
